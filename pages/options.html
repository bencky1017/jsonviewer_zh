  • "prependHeader" ： 显示时间戳和当前URL的注释头（布尔值），默认为 true
  • "maxJsonSize" ： 格式化JSON的最大容量大小，以KB为单位的值，超过将显示提醒（整数）,默认 400
  • "alwaysFold" ： 总是折叠第二层，如果值为 true，加载出来的文档默认折叠第二层（布尔值），默认 false
  • "alwaysRenderAllContent" ： 总是渲染所有内容。请注意，使用此选项大文件将导致性能显着降低 （布尔值）。默认 false
  • "sortKeys" ： 按JSON的Key来进行排序（布尔值），默认 false
  • "clickableUrls" ： 使 URL 可点击（布尔值），默认 true
  • "wrapLinkWithAnchorTag" ： 使用浏览器默认的URL锚点标记样式（布尔值），默认 false
  • "openLinksInNewWindow" ： 在新窗口/新选项卡中打开 URL（布尔值），默认 true
  • "autoHighlight" ： 自动执行格式化高亮，当设置为 false 时，仍然可以点击右侧功能按钮格式化 JSON（布尔值）,默认 true

  • "readOnly" - 只读显示，当为 false 时候，可以对JSON文档进行编辑（布尔值），默认 true
